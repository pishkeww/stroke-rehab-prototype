This project is intended as a post-stroke rehabilitation support system. It uses MediaPipe based computer vision to analyse posture and gross motor movement in real time.
The current implementation uses a standard laptop webcam. However, the system  can be extended to support external cameras or mobile phone cameras with minimal modification.


Note :This is not a diagnostic or clinical assessment system
Validation on post-stroke individuals is planned as future work. After validation, the system will be extended into a Windows application.
